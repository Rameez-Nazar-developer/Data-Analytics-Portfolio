# HR Analytics Dashboard & BI Project

## 📌 Project Overview

This project explores employee workforce data to analyze employee demographics, attrition, job satisfaction, and workforce characteristics using multiple data analytics and business intelligence tools.

The project was completed as a hands-on learning and portfolio project involving Microsoft Excel, PostgreSQL, Power BI, and Tableau.

The same HR dataset was explored across multiple tools to understand how data preparation, analysis, visualization, dashboard development, and validation can be connected within an analytics workflow.

---

## 🎯 Project Objectives

The main objectives of this project were to:

- Analyze overall workforce characteristics.
- Understand employee attrition patterns.
- Examine attrition across departments and demographic groups.
- Analyze employee distribution by age and education.
- Explore job satisfaction across different job roles.
- Develop interactive dashboards using Power BI and Tableau.
- Practice SQL-based analysis of HR metrics.
- Compare analytical results with dashboard outputs through validation exercises.
